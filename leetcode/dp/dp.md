## DP Fundamentals

## Categories

| Class          | Core idea                       |
| -------------- | ------------------------------- |
| 1D / 2D DP     | Prefixes, sequences             |
| Knapsack DP    | Capacity / count tradeoffs      |
| Interval DP    | Subarrays, ranges               |
| Tree DP        | Parent–child structure          |
| Graph DP       | DAGs, shortest paths            |
| **Profile DP** | Small width, row states         |
| Digit DP       | DP over digits with constraints |
