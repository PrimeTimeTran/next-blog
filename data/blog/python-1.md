```python
animals = ['cat', 'dog', 'bat']


for i in range(len(animals)):
  print(animals[i])
```

Prefer using

```python
animals = ['cat', 'dog', 'bat']

for animal in animals:
  print(animal)
```

```python
student = { 'name': 'Harry', 'house': 'Gryffindor' }

  print(student['name'])
```
