# 📘 Calculus 3 Hub

## 🧠 Purpose

Calculus 3 extends calculus into **higher dimensions**.

It studies:

- functions of multiple variables
- geometry in 3D space
- [[vector-fields|Vector Fields]]
- surface and volume integration
- fundamental theorems connecting all of them

---

## 🧠 Core Idea

> Calculus 3 is the study of **change in space**

Instead of:

- x → y (1D)

We now study:

- (x, y, z) → values + vectors + fields

# 📌 Key Insights

- Space introduces direction, not just magnitude
- Gradients generalize derivatives into vectors
- Integrals become area/volume in arbitrary regions
- Vector theorems unify multiple integration methods
- Coordinate systems are tools for simplifying geometry
