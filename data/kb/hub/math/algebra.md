# Algebra Hub

## Purpose

This hub organizes the core concepts, techniques, and problem-solving strategies of algebra. Algebra is the language of mathematics, providing tools to represent relationships, solve equations, and manipulate symbolic expressions.

---

## Core Idea

Algebra is about finding unknown values and understanding relationships between quantities through symbols, equations, and systematic manipulation.

---

## Foundational Concepts

### Variables and Expressions

- **Variables** — Symbols representing unknown or changing quantities
- **Constants** — Fixed numerical values
- **Expressions** — Combinations of variables, constants, and operations
- **Terms** — Individual components separated by + or -
- **Coefficients** — Numbers multiplying variables

### Operations and Properties

- **Commutative Property** — a + b = b + a; ab = ba
- **Associative Property** — (a + b) + c = a + (b + c)
- **Distributive Property** — a(b + c) = ab + ac
- **Identity Elements** — 0 for addition, 1 for multiplication
- **Inverse Elements** — Negatives and reciprocals

### Exponents and Radicals

- **Exponent Rules** — Product, quotient, power rules
- **Zero and Negative Exponents** — a⁰ = 1, a⁻ⁿ = 1/aⁿ
- **Rational Exponents** — a^(m/n) = ⁿ√(aᵐ)
- **Radical Simplification** — Simplifying roots and rationalizing
- **Scientific Notation** — Representing very large/small numbers

---

## Equations and Inequalities

### Linear Equations

- **One-variable**: ax + b = c; solve for x
- **Multi-step**: Combine like terms, isolate variable
- **Two-variable systems**: Substitution, elimination, graphing
- **Applications**: Word problems, mixture problems, motion problems

### Quadratic Equations

- **Standard form**: ax² + bx + c = 0
- **Solving methods**:

  - Factoring: (x - r₁)(x - r₂) = 0
  - Quadratic formula: x = (-b ± √(b² - 4ac))/(2a)
  - Completing the square
  - Graphing

- **Discriminant**: b² - 4ac determines real/complex roots
- **Vieta's formulas**: Relationships between roots and coefficients

### Inequalities

- **Linear inequalities**: ax + b > c
- **Compound inequalities**: AND/OR relationships
- **Quadratic inequalities**: ax² + bx + c > 0
- **Rational inequalities**: f(x)/g(x) < 0
- **Absolute value inequalities**: |x| < a, |x| > a

## Polynomial Operations

### Polynomial Structure

- **Degree** — Highest exponent
- **Leading term** — Term with highest degree
- **Standard form** — Descending order of exponents
- **Types** — Monomial, binomial, trinomial, polynomial

### Polynomial Arithmetic

- **Addition/Subtraction** — Combine like terms
- **Multiplication** — Distributive property, FOIL for binomials
- **Division** — Long division, synthetic division
- **Special products**:
  - (a + b)² = a² + 2ab + b²
  - (a - b)² = a² - 2ab + b²
  - (a + b)(a - b) = a² - b²

### Polynomial Factoring

- **Common factor** — GCF of all terms
- **Grouping** — Factor by grouping pairs
- **Trinomial factoring** — ax² + bx + c = (mx + p)(nx + q)
- **Special forms**:
  - Difference of squares: a² - b² = (a + b)(a - b)
  - Perfect squares: a² + 2ab + b² = (a + b)²
  - Sum/difference of cubes: a³ ± b³

### Polynomial Analysis

- **Rational roots theorem** — Possible rational roots
- **Remainder theorem** — P(a) is remainder of P(x)/(x - a)
- **Factor theorem** — (x - r) is factor iff P(r) = 0
- **End behavior** — Leading term determines long-term behavior

## Rational Expressions

### Operations

- **Simplification** — Cancel common factors
- **Multiplication/Division** — Multiply numerators/denominators
- **Addition/Subtraction** — Common denominator required
- **Complex fractions** — Simplify nested fractions

### Equations and Inequalities

- **Rational equations** — Solve and check for extraneous solutions
- **Rational inequalities** — Sign analysis on intervals

### Asymptotes

- **Vertical asymptotes** — Where denominator = 0
- **Horizontal asymptotes** — End behavior comparison of degrees
- **Oblique asymptotes** — Long division gives linear approximation

## Exponential and Logarithmic Functions

### Exponential Functions

- **Form**: f(x) = aᵇˣ where b > 0, b ≠ 1
- **Properties**: Always positive, domain all reals
- **Transformations**: Vertical/horizontal shifts and stretches
- **Applications**: Growth, decay, compound interest

### Logarithmic Functions

- **Definition**: log_b(x) = y means bʸ = x
- **Common and natural logs**: log₁₀ and ln
- **Properties**:
  - log_b(xy) = log_b(x) + log_b(y)
  - log_b(x/y) = log_b(x) - log_b(y)
  - log_b(xⁿ) = n·log_b(x)
- **Change of base**: log_b(x) = log_a(x)/log_a(b)

### Exponential and Logarithmic Equations

- **Exponential equations**: Take log of both sides
- **Logarithmic equations**: Exponentiate both sides
- **Applications**: Half-life, population growth, pH, decibels

## Sequences and Series

### Sequences

- **Arithmetic sequences**: Constant difference d
  - nth term: aₙ = a₁ + (n-1)d
- **Geometric sequences**: Constant ratio r
  - nth term: aₙ = a₁·rⁿ⁻¹
- **Recursive sequences**: Each term defined by previous terms

### Series (Sums)

- **Arithmetic series**: Sum = n/2(first + last)
- **Geometric series**: Sum = a₁(1 - rⁿ)/(1 - r)
- **Infinite geometric series**: Sum = a₁/(1 - r) if |r| < 1
- **Summation notation**: Σ and properties

## Systems of Equations

### Linear Systems

- **Two variables**: Graphing, substitution, elimination
- **Three variables**: Elimination method, Gaussian elimination
- **Matrices**: Represent and solve using row operations
- **Consistency**: Unique solution, no solution, infinitely many

### Non-linear Systems

- **Quadratic-linear combinations**: Parabola intersecting line
- **Quadratic-quadratic**: Two parabolas intersecting
- **Solution methods**: Substitution, elimination adapted

---

## Common Problem Types

### Solving Strategies

1. **Linear equations**: Isolate variable using inverse operations
2. **Quadratic equations**: Factor or use quadratic formula
3. **Systems**: Choose method based on structure (substitution, elimination)
4. **Inequalities**: Solve equation first, test intervals
5. **Rational expressions**: Find common denominator or LCD
6. **Exponential/logarithmic**: Take log or exponentiate appropriately

### Word Problem Translation

- Identify variable(s) and what they represent
- Set up equation(s) from relationships described
- Solve algebraically
- Check solution makes sense in context

## Key Theorems and Facts

- **Fundamental Theorem of Algebra** — Polynomial degree n has exactly n roots (counting multiplicity)
- **Rational Root Theorem** — Possible rational roots are ±(factors of constant)/(factors of leading coefficient)
- **Descartes' Rule of Signs** — Number of positive roots related to sign changes
- **Intermediate Value Theorem** — Continuous function takes all intermediate values

---

## Common Mistakes

- Distributing incorrectly (forget to distribute to all terms)
- Sign errors (especially with negatives and subtraction)
- Dividing by zero or zero in denominator
- Extraneous solutions from equations (check!)
- Reversing inequality when multiplying by negative
- Combining terms incorrectly (only combine like terms)
- Factoring errors (expand to check)

## Key Insights

- **Inverse operations**: Addition/subtraction, multiplication/division undo each other
- **Factoring**: Reverse of multiplication; essential for solving
- **Exponent rules**: Extension of repeated multiplication to general operations
- **Logarithms**: Inverse of exponentials; convert multiplication to addition
- **Symmetry**: Polynomials, even/odd functions, inverse functions
- **Patterns**: Many problems fit recognizable forms with standard solution methods
