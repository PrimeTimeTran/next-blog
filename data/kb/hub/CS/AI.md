# Artificial Intelligence (AI)

## Overview

Artificial Intelligence is the field of building systems that can perform tasks typically requiring human intelligence—such as reasoning, learning, perception, language understanding, and decision-making.

---

## Core Subfields

### [[atom/CS/AI/ML|Machine Learning]]

Systems that learn patterns from data rather than being explicitly programmed.

- [[atom/CS/AI/ML/core/supervised/overview|Supervised]] (labeled data)
- [[atom/CS/AI/ML/core/unsupervised/overview|Unsupervised]] (structure discovery)
- [[atom/CS/AI/ML/core/reinforcement/overview|Reinforcement]] (learning via reward signals)
- [[atom/CS/AI/ML/core/deep-learning/overview|Deep]] (neural networks): A subset of ML using neural networks with many layers.
  - CNNs → images
  - RNNs / Transformers → sequences, language
  - Foundation models → large pre-trained general-purpose systems

### Natural Language Processing (NLP)

Understanding and generating human language.

- Translation
- Chatbots
- Text classification
- Summarization

### Computer Vision

Interpreting visual data.

- Object detection
- Image classification
- Segmentation

### Reinforcement Learning (RL)

Learning through interaction and reward feedback.

- Agents, environments, policies
- Exploration vs exploitation

---

## Key Concepts

- **Data** → fuel for learning systems
- **Model** → learned function mapping inputs → outputs
- **Training** → adjusting model parameters using data
- **Inference** → using trained model to make predictions
- **Loss function** → measures prediction error
- **Optimization** → minimizing loss (e.g., gradient descent)

---

## AI System Stack

1. Data collection & preprocessing
2. Model architecture
3. Training process
4. Evaluation & validation
5. Deployment & inference
6. Monitoring & iteration

---

## Types of AI

- **Narrow AI** → task-specific systems (most real-world AI today)
- **General AI (AGI)** → human-level general intelligence (not yet achieved)
- **Superintelligence** → hypothetical systems beyond human intelligence

---

## Common Applications

- Search engines
- Recommendation systems (YouTube, Netflix)
- Autonomous vehicles
- Medical diagnosis
- Voice assistants
- Fraud detection
- Generative AI (text, image, code generation)

---

## Big Ideas

- Intelligence emerges from patterns + data + optimization
- Better models = better representations of reality
- Scaling data + compute often improves performance
- Generalization is harder than memorization
- Tradeoffs exist: accuracy vs interpretability vs compute cost
