3D CAD software doesn't have a single universal file format, but there are several commonly used formats that are widely supported across different CAD programs. Here are some of the most popular ones:

1. **STL (Stereolithography)**: Commonly used for 3D printing, it's widely supported but doesn't contain information about colors, materials, or other attributes.

2. **OBJ (Wavefront Object)**: Supports 3D geometry and can include texture information, making it suitable for 3D modeling and animation.

3. **STEP (Standard for the Exchange of Product model data)**: An ISO standard (ISO 10303) that is widely used for exchanging CAD data between different software programs. It supports complex assemblies, colors, and other attributes.

4. **IGES (Initial Graphics Exchange Specification)**: Another widely supported format for exchanging 3D models. It can represent both surface and solid models.

5. **3MF (3D Manufacturing Format)**: Developed by the 3MF Consortium, this newer format is designed for 3D printing and includes more comprehensive information than STL.

6. **FBX (Filmbox)**: Developed by Autodesk, it is used for 3D animation and supports complex data including textures, lighting, and more.

Each of these formats has its own strengths and use cases, so the choice of format often depends on the specific requirements of the project and the software being used.

Fusion 360 primarily uses the **F3D** and **F3Z** file formats for its native files:

- **F3D**: This is the standard file format for a single design file in Fusion 360. It includes all the design data, such as components, sketches, timelines, and parameters.
- **F3Z**: This format is used for Fusion 360 archive files and can include multiple F3D files along with other related data.

In addition to these native formats, Fusion 360 supports a wide range of other file formats for import and export, including:

- **STEP (STP)**
- **IGES (IGS)**
- **STL**
- **OBJ**
- **DXF**
- **FBX**
- **SAT**
- **SMT**
- **SketchUp (SKP)**
- **SolidWorks (SLDPRT, SLDASM)**
- **Parasolid (X_T, X_B)**
- **Inventor (IPT, IAM)**

These formats enable interoperability with other CAD software and facilitate the sharing and collaboration of designs.
