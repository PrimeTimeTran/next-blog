### First Derivative Test

The First Derivative Test is a method for determining **local maxima and minima** by analyzing how a function’s slope changes.

---

## **Core Idea**

A function’s **extrema occur where its derivative changes sign**.

- ( f'(x) > 0 ) → function is increasing
- ( f'(x) < 0 ) → function is decreasing

---

## **Critical Points**

Find points where:

$$
f'(x) = 0 \quad \text{or undefined}
$$

These are the only places where a local max or min can occur.

---

## **First Derivative Test Rule**

At a critical point ( c ):

### 1. Local Maximum

If:

- ( f'(x) > 0 ) before ( c )
- ( f'(x) < 0 ) after ( c )

Then:

> ( f ) has a **local maximum at ( c )**

---

### 2. Local Minimum

If:

- ( f'(x) < 0 ) before ( c )
- ( f'(x) > 0 ) after ( c )

Then:

> ( f ) has a **local minimum at ( c )**

---

### 3. No Extremum

If the sign does not change:

> No local max or min at ( c )

---

## **Intuition**

- Positive derivative → climbing uphill
- Negative derivative → going downhill
- Extremum happens where direction switches

---

## **Workflow**

1. Compute ( f'(x) )
2. Solve ( f'(x) = 0 )
3. Test intervals around critical points
4. Check sign changes

---

## **One-line summary**

The First Derivative Test classifies local maxima and minima by tracking whether ( f'(x) ) changes from positive to negative or vice versa.
