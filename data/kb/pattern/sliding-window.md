# Pattern: Sliding Window

## When to Use

- Contiguous subarray/substring
- Optimize from brute force
- Constraint-based window (sum, distinct chars, etc.)

## Template

1. Initialize window
2. Expand right
3. Shrink left when constraint breaks

## Variants

- Fixed window
- Variable window

## Example Problems

- Longest substring without repeating characters
