# Bushido

A samurai moral code concerning samurai attitudes, behavior, and lifestyle.
Also used as an overarching term for all the codes, practices, philosophies, and principles of samurai culture.

**Stillness —** 5 min Mokuso meditation

**Breath —** Box breathing technique. 4 seconds each for breathing in, holding, out, pausing breathes

**Movement —** Full body mobility flow

**Strength —** Ancient bodyweight training: 20 pushups, 20 squats, 20 second deadhang, 45 sec plank

**Purpose —** The warrior's daily intention: Set a single behavior intention for the day. A single word like patience, courage, or growth.
Target for mental calm and physical strength are useful is they're aimed at a single target.
